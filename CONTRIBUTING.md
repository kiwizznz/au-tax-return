# Contributing to AU Tax Return

Thanks for your interest in contributing! This project is a Claude Cowork plugin that helps Australians with their tax returns.

## How to Contribute

### Reporting Issues

- Use [GitHub Issues](https://github.com/kiwizznz/au-tax-return/issues) to report bugs or suggest features.
- Include steps to reproduce any bugs.
- For tax-related suggestions, specify the relevant ATO form or schedule.

### Submitting Changes

1. Fork the repository.
2. Create a feature branch from `main`.
3. Make your changes.
4. Submit a pull request against `main`.

### What We're Looking For

- Improvements to tax guidance accuracy (aligned with current ATO rules).
- Better handling of specific tax scenarios (rental properties, capital gains, crypto, etc.).
- Clarity and usability improvements to prompts and instructions.
- Bug fixes.

## Important Notes

- **This is not tax advice.** This tool helps users organise information for their tax returns. It does not replace a registered tax agent.
- **Accuracy matters.** If you're contributing tax-related logic or guidance, please reference the relevant ATO documentation.
- **Keep it current.** Australian tax rules change each financial year. Contributions should target the current or upcoming FY.

## Code of Conduct

Be respectful and constructive. We're all here to make tax time a bit less painful.
